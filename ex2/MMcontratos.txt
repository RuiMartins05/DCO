2º Caso de Uso: Procurar Ajuda
Ator Principal: Migrante

Pré-Condições: N/A

Pós-Condições: Registadas ajudas no sistema e sms enviado aos voluntarios das ajudas.

--

CONTRATOS: 

    op: registaMigrante(nome, contacto)
        pré-condições:
            - N/A.
        pós-condições:
            - m:Migrante esta autenticado e é corrente;
            - m.nome = nome e m.contacto = contacto. 

    op: registaFamilia(num)
        pré-condições:
            - num > 1;
        pós-condições:
            - m:Migrante esta autenticado e é corrente;
            - m.numPessoasFam = num;
            - é criado fam:String[num]. 

    op: indicaCabecaCasal(nome, contacto)
        pré-condições:
            - m:Migrante esta autenticado.
        pós-condições:
            - m.nome = nome e m.contacto = contacto.
            
    op: indicaMembro(nome)
        pré-condições:
            - m:Migrante esta autenticado.
        pós-condições:
            - nome é adicionado a fam.

    op: getListaRegioes()
        pré-condições:
            - m:Migrante esta autenticado;
            - existe lr:ListaRegioes.
        pós-condições:
            - N/A.

    op: indicaRegiao(regiao)
        pré-condições:
            - m:Migrante esta autenticado;
            - existe lr:ListaRegioes;
            - la:ListaAjudas associada a regiao.
        pós-condições:
            - N/A.

    op: indicaAjuda(la)
        pré-condições:
            - m:Migrante esta autenticado;
            - existe la:ListaAjudas;
            - criada (ajudasSelecionadas:list<Ajuda>) se ajudasSelecionadas == null. 
        pós-condições:
            - seja a:Ajuda tal que a:Ajuda pertence a la:ListaAjudas, m.ajudaCorrente = a.

    op: confirmarAjudas()
        pré-condições:
            - m:Migrante esta autenticado;
            - existe ajudasSelecionadas:list<Ajuda> corrente.
        pós-condições:
            - pretendeConfirmar = true.

- Extensão 5a:

    op: pretendeNotificacao(regiao)
        pré-condições:
            - m:Migrante esta autenticado;
            - existe lr:ListaRegioes empty;
            - existe pretendeNotificacao:list<Migrante>.
        pós-condições:
            - migrante associado a regiao;
            - adicionado migrante a pretendeNotificacao.
